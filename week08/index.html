Week 08 Notes & Exercises

Notes

  • You can use the CSS3 transform property to alter the location of an element. You can use transform: translate(45px, -45px) to move an element to (x, y). Note: transforms don't work on inline elements. Change it to inline-block, etc. You can put multiple transform properties under the same transform (ex: both translate and scale).
  • Transform: scale(x,y) will scale your element bigger or smaller. Just doing scale(2) would double its size, whereas doing scale(1, 0.25) would keep the horizontal scale the same but then shrink it vertically by 0.25.
  • Here is a list of all the transform properties mentioned in chapter 8: translate(x,y); translateX(x); translateY(y); scale(x,y); rotate(_deg); skew(x,y).
  • You can change the transform-origin to 0 0 or top center, etc, to change the location on the element where it's rotated, etc.
  • Note: the order your transform elements matters. Otherwise you will end up with an element in a different location or in the wrong shape.
  • You can use the CSS3 transition property to change an element over time. Use transition-property to define which css properties should be transitioned (all is default). Use transition-duration to set how long the transition should take (s or ms). Use transition-timing-function to control the pace of the transition (start slow, end fast, vice versa). Use transition-delay to delay when the transition starts (s or ms). You can also just use transition and list our the four transitions.
  • You can use animations to have more control over your transitions. Animations use keyframes, which are essentially little snapshots that guide the direction of the animation.Use animation-name, animation-duration, animation-iteration-count, animation-direction, animation-delay, animation-fill-mode, animation-play-state and animation-timing-function to create your animation.
  • Canvas is an HTML way to draw anything through javascript. This is a good method if you want to create and save images or manipulate existing pixels. But you can't edit an existing object in a canvas without starting over.
  • Scalable Vetor Graphics (SVG) are file formats than can be shrunk or enlarged and preserve the quality. This is a good method if you want to access what you draw through a DOM, use more complex tools, and edit an existing SVG object. However, you can't manipulate the pixels of an SVG.
  • You can make elements of your page draggable using the draggable attribute, and you can create event listeners for the dragstart, dragover, and drop events on other elements.
