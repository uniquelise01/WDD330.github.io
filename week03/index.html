Week 03 Notes & Exercises

Notes

  • To my understanding, object literals are objects that contain multiple values and/or functions. You can also create nested objects.
  • It's interesting that changes to a variable assigned to an already-existing object will change bothe the object and the new variable. However, if you use primitive values, like letters, it won't do the same thing.
  • The this method can sometimes confuse me, but the more I study it, the more it just seems like this is just referencing the original object. I could be wrong, but sometimes things are simpler than they seem.
  • To avoid repeating object, object literal, or function names, try using name spacing: put all of your functions and objects inside the same object literal, like const myMaths = {}. Then, when called, just use myMaths.square(x), etc.
  • JSON (JavaScript Object Notation) is a format we use to configure and serialize data, in a similar format as object literals. However, in JSON, property names need to be in double-quotes and functions are not permitted. To turn JSON into JS, use .parse(). To convert JS into JSON, use .stringify().
  • There are several very useful math objects in JS. I'll put them in a table:
    • Math.PI Holds the value of pi
    Math.SQRT2 The value of the square root of 2
    Math.E Euler's constant
    Math.LN10 Natural log of 10
    Math.LOG10E Log base 10 of Euler's constant
    Math.abs() Absolute value of a number
    Math.ceil() Rounds up to the nearest integer
    Math.floor() Rounds down to the nearest integer
    Math.round() Rounds to the nearest integer
    Math.trunc() Removes the decimal of a number
    Math.exp() Raises a number to the power of Euler's constant
    Math.pow( , ) Raises the first number to the power of the second number
    Math.sqrt() Returns the positive square roots
    Math.cbrt() Finds the cube root of a number
    Math.log() Returns the natural log of a number
    Math.max( ,...) Returns the maximum number you entered
    Math.min( ,...) Returns the minimum number you entered
    Math.sin() Returns the sine of an angle
    Math.cos() Returns the cosine of an angle
    Math.tan() Returns the tangent of a number
    Math.random() Generates a random number between 1 and 0.
    x * Math.random() Generates a random number between x and 0.
  • You can use regular expressions as a kind of "Find and replace" code. Ex: const pattern = new RegExp('[a-zA-Z]+ing'); will return true if a word has 'ing' at the end of it.
