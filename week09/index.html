Week 09 Notes & Exercises

Notes

  • The window object is the global object for a browser. You can use it to do things like find the screen dimensions, create pop-up windows, create alerts, create confirm pop-ups, create prompts, etc.
  • You can create global variables without using const, let, or var. You just declare them, like x = 6.
  • The window.navigator.userAgent can tell you what browser and operating system is being used. You can use window.location.href to identify the current page url, and you can also set it = to something to redirect someone to a different page. There's also location.protocol, .host, .hostname, .port, and a bunch of other location properties.
  • Use window.history to access previously visited pages in the current browser session. There are a few properties that go with .history as well.
  • You can also use window.moveTo(x,y) and window.resizeTo(px,px) but you should usually avoid these since they're really annoying. You can get the screen width and height with window.screen.height, etc.
  • Cookies allow information to be stored between visits, so browsers can retrieve information for later visits. You can create them by doing document.cookie = 'string=ofInformation; name=Superman; hero=true'.
